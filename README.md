# Gestao Financeira Casa e Empresa

Abra no computador pelo servidor local:

http://127.0.0.1:5177/index.html

Login inicial:

- Usuario: admin
- Senha: 1234

O sistema salva os dados no navegador usando localStorage. Ele possui painel mensal, contas Casa/Empresa, status Pago/A vencer/Vencida, cartoes, lembretes e exportacao CSV.

Para instalar como aplicativo no celular, publique esta pasta em uma hospedagem com HTTPS ou peça para transformar em app com backend/login real. Depois abra o link no celular e use a opcao "Adicionar a tela inicial" do navegador.
